# Rev. Prince Chukwuma Ministries — Official Website

Official website for Ven. Prince Chukwuma (Papa Umuboyii), Archdeacon in the Anglican Communion, Diocese of Nnewi. Built to showcase his ministry, fellowships, business academy, and event solutions.

---

## Pages

- **Home** — Hero, About, and Explore section linking to all ministry arms
- **RCBA** — Resource Christian Business Academy & Global Business and Prayer Conference
- **Umuboyii** — Umuboyii Na Ndi Oga Fellowship page
- **PowerSound** — PowerSound System event equipment hire business
- **Ministry** — Ministry overview, teachings, gallery, and social media links

---

## Built With

- React.js
- React Router DOM
- React Icons
- CSS (custom styling per page)

---

## Features

- Fully responsive design (mobile, tablet, desktop)
- Fixed navigation with active link highlighting
- Image galleries across multiple pages
- Embedded YouTube videos
- WhatsApp and email booking links for PowerSound
- Social media integration (Facebook, Instagram, YouTube, TikTok)
- Scroll to top on page navigation
